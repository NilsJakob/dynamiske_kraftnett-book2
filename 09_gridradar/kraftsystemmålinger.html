SCADA data er operasjonell data og ofte propietær og man må ha spesiell tilgang for å aksessere denne type data. IEEE DataPort er en online plattform der forskere og ingeniører kan dele, finne og bruke datasett. Det er sjeldent at man finner ekte SCADA data her av grunner nevnt ovenfor men på GitHub, en plattform for å lagre, dele og samarbeide på kode og prosjekter, er noe tilgjengelig på OpenWindSCADA. Github ble laget av utviklere i 2008 og kjøpt av Microsoft i 2018. OpenWindSCADA samler lenker til andre dataplattformer som til open-source plattformen Zenodo initiert av partikkel-fysikk laben CERN i Sveits og støttet av EU. På Zenodo kan man laste ned åpen SCADA data fra Kelmarsh windfarm

Du kan laste ned datasettet her:

Phasor Measurement Units#

En Phasor Measurement Unit (PMU) er et måleinstrument i kraftsystemet som måler spenning og strøm med både størrelse og fasevinkel, synkronisert i sanntid.

En PMU er et avansert måleinstrument som måler strømnettet i sanntid med svært høy presisjon og synkronisert med GPS-klokke.

En PMU måler typisk:

  • Spenningsnivå (V)

  • Spenningsvinkel (\(\delta\))

  • Strøm (I)

  • Strømfase (\(\theta\))

  • Frekvens (Hz)

  • Frekvensforandring (\(\frac{df}{dt}\) RoCoF -rate of change of frequency)

Målinger over hele Europa skjer på akkurat samme tidspunkt med nøyaktighet på mikrosekund-nivå.

Synkroniserte målinger gjør det mulig å:

  • måle faseforskjeller mellom områder

  • forstå hvordan effekt flyter i nettet

  • oppdage forstyrrelser umiddelbart

Sted

Fasevinkel

Oslo

10°

Berlin

5°

Forskjell = 5° → betyr at effekt flyter fra Oslo til Berlin

PMU målinger leveres av mainsfrequency og gridradar, sistnevnte med eget API.

import os
import requests
from dotenv import load_dotenv
import pandas as pd

load_dotenv()

token = os.getenv("GRIDRADAR_TOKEN")

url = "https://api.gridradar.net/query"

headers = {
    "Authorization": f"Bearer {token}",
    "Content-Type": "application/json"
}

data = {
    "metric": "frequency-ucte-median-1s",
    "func": "min",
    "aggr": "1m"
}

response = requests.post(url, headers=headers, json=data)

print("Status:", response.status_code)
print(response.json())
